# Reference Screenshots

Use this folder for AppFolio/reference screenshots that Portier should copy or map.

## How to save a screenshot

1. Take a screenshot.
2. Save it as a PNG in `docs/reference-screenshots/images/`.
3. Use this naming pattern:

```text
YYYY-MM-DD-leftnav-page-name.png
```

Examples:

```text
2026-05-12-associations-directory.png
2026-05-12-associations-new.png
2026-05-12-dashboard.png
```

## How Codex will use it

Each screenshot should be logged in `docs/reference-screenshots/screen-map.md` with:

- left pane selection
- center pane contents
- right pane tasks
- Supabase tables/actions needed
- status

This keeps the project grounded in approved screenshots instead of memory.
