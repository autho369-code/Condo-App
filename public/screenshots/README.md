# Marketing screenshots

Drop PNG screenshots here with these EXACT filenames. The landing page
auto-detects them — if a file is missing, it falls back to the built-in
placeholder, so nothing breaks.

## How to capture (Windows)
1. Open the page in Chrome at a normal window size (~1440px wide).
2. Press Win + Shift + S, drag a clean rectangle around the content area
   (no browser chrome, no bookmarks bar).
3. It copies to clipboard — paste into Paint, Save As PNG with the name below.
   (Or set Snipping Tool to auto-save.)
